# 📊 Order Streamlit Dashboard

Интерактивный аналитический дашборд, разработанный на **Streamlit**, для анализа данных интернет-магазина.

## Возможности

Приложение позволяет:

- загружать данные из CSV-файлов;
- объединять данные о заказах, пользователях и товарах;
- выполнять очистку и подготовку данных;
- рассчитывать ключевые бизнес-показатели;
- визуализировать основные метрики;
- формировать аналитические выводы.

# Блоки дашборда

## 📋 Сырые данные

Отображается исходный объединенный DataFrame.

Доступны:

- просмотр данных;
- фильтрация по периоду;
- фильтрация по категории товара.

---

## 📈 Ключевые показатели

Рассчитываются основные бизнес-метрики:

- общее количество заказов;
- общая выручка;
- количество уникальных пользователей;
- средний чек.

---

## 📊 Визуализация

Дашборд содержит несколько аналитических графиков.

### Топ-10 товаров по выручке

Горизонтальная столбчатая диаграмма. Показывает товары, принесшие наибольшую выручку.

---

### Выручка по категориям

Круговая диаграмма отображает распределение выручки между категориями товаров.

---

### Количество заказов по дням недели

Столбчатый график показывает активность покупателей в разные дни недели.

---

## 📌 Аналитические выводы

На основе рассчитанных показателей автоматически формируются выводы:

- категория с максимальной выручкой;
- товар-лидер по продажам;
- день недели с наибольшим количеством заказов

# Запуск проекта

## 1. Установить зависимости

```bash
pip install -r requirements.txt
```

## 2. Запустить приложение

```bash
streamlit run app.py
```

После запуска приложение будет доступно по адресу:

```
http://localhost:8501
```
